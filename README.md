# Bidarur Event Management Website

A static event management website created using only HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - Homepage with hero section, events, schedule, and registration form
- `about-project.html` - Project information page
- `css/style.css` - Main styling
- `css/responsive.css` - Responsive layout rules
- `js/script.js` - Mobile menu toggle and form validation
- `images/` - Folder for event images or logos

## Features

- Responsive layout for desktop and mobile
- Attractive event landing page
- Registration form with JavaScript validation
- Separate About Project page

## Run the Project

Open `index.html` in your browser.
